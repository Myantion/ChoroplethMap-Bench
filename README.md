# ChoroplethMap-Bench
ChoroplethMap-Bench: A benchmark for evaluating the cognitive compression and spatial reasoning of MLLMs using 2,400 synthetic choropleth maps.

Citation for the data and code: Wei, Z., Sun, Y., Liu,Z., Xu, W., Dong, W., Liu, C., Liao, H., 2026. ChoroplethMap-Bench: A benchmark for evaluating the cognitive compression and spatial reasoning of MLLMs using 2,400 synthetic choropleth maps. https://github.com/Myantion/ChoroplethMap-Bench.
